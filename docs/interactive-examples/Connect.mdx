# Connect

Connect your dapp to Algorand users's wallets.
User should grant access and choose the wallets that wish to share with your dapp.

### Preconditions

* The user must have MyAlgo Wallet Account previously created in **[MyAlgo Wallet](https://wallet.myalgo.com)** to be able to connect to the dapp.

* User should have a wallet previously created in **[MyAlgo Wallet](https://wallet.myalgo.com/home)** in order to be able to share it.
